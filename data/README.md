# Data

The raw data file (`avall.mdb`) is not committed to this repository due to file size.

## How to get the data

1. Go to https://www.ntsb.gov/safety/data/Pages/Data_Stats.aspx
2. Download `avall.zip` under "Download the full dataset"
3. Extract `avall.mdb` into this folder
4. Run `notebooks/01_data_loading.ipynb` to generate `aviation.db`
